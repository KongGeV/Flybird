# 愤怒的小鸟游戏

这是一个使用HTML5 Canvas和JavaScript开发的简易版愤怒的小鸟游戏。

## 游戏玩法

1. 点击"开始游戏"按钮开始
2. 点击屏幕或按空格键使小鸟跳跃
3. 避开绿色管道障碍物
4. 每通过一个障碍物得1分
5. 碰到障碍物或地面游戏结束

## 如何运行

直接在浏览器中打开index.html文件即可开始游戏。

## 文件结构

- `index.html`: 游戏的HTML结构
- `style.css`: 游戏的样式
- `script.js`: 游戏的JavaScript逻辑

## 技术细节

- 使用HTML5 Canvas进行游戏绘制
- 使用requestAnimationFrame进行游戏循环
- 实现了简单的物理引擎（重力和跳跃）
- 碰撞检测系统
- 响应式设计
